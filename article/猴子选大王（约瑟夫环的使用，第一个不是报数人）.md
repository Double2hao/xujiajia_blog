##  Sample Output

```
#include &lt;iostream&gt;
using namespace std;

int main()
{
  int k,n,m,s=0;
  while(cin&gt;&gt;n&gt;&gt;k&gt;&gt;m)
    {
        for(int i=2;i&lt;=n;i++)
            {
                s=(s+m)%i;
            }

            s=(s+k-1)%n;
            cout&lt;&lt;s+1&lt;&lt;endl;
        }

        return 0;
}
```
