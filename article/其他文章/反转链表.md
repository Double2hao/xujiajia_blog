# 题目：

定义一个函数，输入一个链表的头结点，反转该链表并输出反转后链表的头结点。链表定义如下：

```
struct ListNode
{
    int m_nkey;
    ListNode* m_pNext;
};


```

# 要点

1、链表的建立中由于需要改变头指针指向的值，所以要用双重指针。 2、反转的逻辑主要是让后面结点指向前面的结点，但是如果仅仅这样操作，链表就断了，无法继续向后操作。所以同时需要保存三个结点，前两个结点进行反转操作，后面一个结点用于保证链表的连续性。最主要的逻辑在代码中只是以下这一段：

```
ListNode* Reverse(ListNode* pHead)//链表的反转
{
    ListNode* pHead2=NULL;
    ListNode* pNode=pHead;
    ListNode* pPrev=NULL;
    while(pNode!=NULL)
    {
        ListNode* pNext=pNode-&gt;next;
        if(pNext==NULL)
            pHead2=pNode;

		pNode-&gt;next=pPrev;
        pPrev=pNode;
        pNode=pNext;
    }
    return pHead2;
}

```

#include&lt;iostream&gt;

using namespace std;

struct ListNode
{
    int key;
    ListNode* next;

{
    int n;
    cin&gt;&gt;n;

    ListNode* pNode=NULL;
    ListNode* pPrev=NULL;

    int k;
    cin&gt;&gt;k;
    pNode=new ListNode;

	*pHead=pNode;
    pPrev=pNode;

    while(--n)
    {
        cin&gt;&gt;k;
        pNode=new ListNode;
        pNode-&gt;key=k;

        pPrev-&gt;next=pNode;
        pPrev=pNode;
    }
    pNode-&gt;next=NULL;
}

{
    ListNode* pNode=pHead;
    while(pNode!=NULL)
	{
        cout&lt;&lt;pNode-&gt;key&lt;&lt;endl;
		pNode=pNode-&gt;next;
	}
}

{
    ListNode* pHead2=NULL;

    ListNode* pPrev=NULL;
    while(pNode!=NULL)

            pHead2=pNode;

		pNode-&gt;next=pPrev;

}
